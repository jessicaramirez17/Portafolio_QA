"¡Hola! Bienvenido a mi portafolio"


Mi nombre es Jessica, durante mi proceso de formación profesional siempre estuve muy interesada por el área de aseguramiento de la calidad del software (QA). Por lo que a través de proyectos académicos y personales, he adquirido experiencia práctica en la creación y ejecución de casos de prueba, el reporte de errores y la validación de funcionalidades, utilizando herramientas como Postman, Jira, TestRail, Azure DevOps y bases de datos MySQL.He trabajado en pruebas sobre APIs públicas y aplicaciones web, lo que ha fortalecido mi capacidad para identificar fallos, colaborar en equipos multidisciplinarios y aplicar metodologías ágiles como Scrum y Kanban, contribuyendo a la entrega de software de calidad.

Proyectos QA

Proyecto QA Manual - API pública RESTful de GoRest usando POSTMAN

Se ejecutaron pruebas manuales a los endpoints públicos de la API GoRest para validar operaciones como creación, consulta y validación de errores en usuarios. Las pruebas se realizaron en Postman, enfocadas en verificar respuestas HTTP, autenticación y manejo de datos inválidos.

Proyecto QA Manual a la Aplicación OrangeHRM


Se realizaron pruebas manuales funcionales sobre la plataforma OrangeHRM para validar módulos como login, administración y gestión de empleados. Se diseñaron y ejecutaron casos de prueba, se documentaron hallazgos y se reportaron bugs

Proyecto QA Manual - aplicación Buggy Cars Rating


Se realizaron pruebas manuales para validar funcionalidades como registro/login, calificación y reseñas de autos, filtros por categoría, edición de perfil y gestión de sesiones. El objetivo fue identificar errores y asegurar el correcto funcionamiento de la aplicación.
